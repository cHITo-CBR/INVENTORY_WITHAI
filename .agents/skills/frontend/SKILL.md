# Frontend Implementation Skill

This skill provides comprehensive instructions for building high-quality, performant, and accessible user interfaces.

## Core Principles
1. **Semantic HTML**: Always use appropriate HTML5 elements (e.g., `<main>`, `<section>`, `<nav>`, `<article>`).
2. **Vanilla CSS**: Prioritize standard CSS over utility frameworks. Use CSS Variables for theme management.
3. **Accessibility (a11y)**: Ensure all interactive elements are keyboard reachable and screen-reader friendly (ARIA labels, alt text).
4. **Performance**: Optimize image loading, use Next.js server components where appropriate, and minimize bundle size.

## Design Rules
- Follow the "Wow" factor: use vibrant colors, glassmorphism, and smooth transitions.
- Use a consistent spacing system (e.g., 4px/8px grid).
- Ensure all interactive elements have unique and descriptive IDs.

## Folder Structure
- `app/`: Next.js App Router pages and layouts.
- `components/`: Reusable UI components.
- `styles/`: Global styles and CSS variables.
