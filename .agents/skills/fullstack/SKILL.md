# Fullstack Development Skill

This skill integrates frontend and backend expertise to deliver complete, functional features.

## Core Principles
1. **Feature-First**: Develop features vertically (from UI to Database) rather than horizontally in silos.
2. **Data Flow Management**: Maintain a clear understanding of how data flows through the entire system.
3. **E2E Testing**: Verify that all system components work together as intended.
4. **Communication**: Ensure the frontend and backend teams (or components) are aligned on data structures and API contracts.

## Key Skills
- State management across the stack (client-side vs. server-side).
- Authentication and authorization integration.
- Asset management (images, static files, CDNs).

## Implementation
- Start by defining the API contract.
- Build the backend to support the contract.
- Implement the frontend to consume the API.
- Validate the integration with end-to-end tests.
